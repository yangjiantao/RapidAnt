## wanandroid module intro
>基于android jetPack实现，api数据来源http://www.wanandroid.com/

## V0.0.1 features
- android jetpack 实现App架构
- 首页推荐文章列表
- 使用Room实现db缓存、Retrofit 加载api接口数据
- 使用androidx 支持库
